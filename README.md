# E-Commerce Sales Dashboard (Power BI)

This project is a complete end-to-end Power BI dashboard built using e-commerce sales data from a women's clothing store. It provides actionable insights into revenue trends, product performance, and customer behavior.

---

## Features

### Data Transformation & Modeling
- Cleaned raw dataset in Power Query
- Converted data types for date, numbers, categories
- Created DAX measures:
  - Total Revenue
  - Total Orders
  - Total Items Sold
  - Average Order Value (AOV)

---

## Dashboard Insights

### KPI Cards
- Total Revenue  
- Total Orders  
- Total Items Sold  
- AOV

### Trend Analysis
- Revenue by Order Date (Daily Trend)

### Product Performance
- Top 10 SKUs by Revenue  
- Revenue by Price (Unit Price)  
- Revenue by Color  
- Revenue by Size  

### Filters (Slicers)
- Date  
- Size  
- Color  

---

## Key Insights
- Identified high-selling SKUs
- Found top-performing product colors & sizes
- Analyzed which price points generate the most revenue
- Daily revenue trend showing order spikes
